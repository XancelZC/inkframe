# Design Guide — InkFrame

基于 Notion 设计系统的 UI 开发指南。

## 视觉核心主张

**温暖的工具感**——这是一个面向小说作者的写作/编辑工具，不是数据仪表盘，不是营销页面。设计应该让作者感到舒适、专注、被尊重。

关键特质：
- **纸质感**：暖色调背景（`#f6f5f4`）营造纸质触感，减少屏幕疲劳
- **安静的结构**：超薄边框和微阴影提供层级，但绝不抢夺注意力
- **内容优先**：所有 UI 元素都为文本内容服务，装饰性元素最少化
- **专业但不冷**：四级字重系统提供丰富层级，暖中性色保持亲切感

## Do's

- 使用暖中性色调——所有灰色都带黄棕底色（`#f6f5f4`、`#31302e`、`#615d59`、`#a39e98`）
- 使用 Inter 字体族，四级字重：400（正文）、500（UI 交互）、600（强调）、700（标题）
- 所有 letter spacing 使用 `0`，保证编辑器和工具界面在不同视口下稳定排版
- 使用超薄边框 `1px solid rgba(0,0,0,0.1)` 作为默认分隔
- 使用多层阴影堆叠（单层透明度 ≤ 0.05）营造微深度
- 使用 `#0075de`（Notion Blue）作为唯一强调色，仅用于 CTA 和链接
- 使用 pill 形状（`9999px` 圆角）用于状态标签和徽章
- 按钮/输入框使用 `4px` 圆角，卡片使用 `8px` 圆角，大型面板和模态使用 `12px` 圆角
- 使用暖白色（`#f6f5f4`）作为交替区块背景，创造视觉节奏
- 保持 `8px` 基础间距单位

## Don'ts

- **禁止使用 Emoji**——所有图标必须使用 SVG 图标或图标库（如 Lucide Icons、Heroicons）。Emoji 在不同平台渲染不一致，破坏视觉统一性
- 不使用纯黑色（`#000000`）作为文字色——使用 `rgba(0,0,0,0.95)` 柔化
- 不使用冷色调灰色（蓝灰）——所有中性色必须带暖底色
- 不使用粗边框——`1px` 是上限
- 不使用高透明度阴影（单层 > 0.05）——阴影应该是"可感知但不明显"
- 不使用 `700` 以下的字重用于正文——正文用 `400`
- 不使用负字间距或正字间距——所有文本统一使用 `letter-spacing: 0`
- 不在按钮上使用 `9999px` 圆角——pill 仅用于标签/徽章
- 不使用渐变色背景——通过暖色调交替实现视觉层次
- 不过度使用动画——过渡时间控制在 150-300ms，仅用于交互反馈

## 组件规范

### 按钮

**Primary**：`#0075de` 背景，白色文字，`4px` 圆角，`8px 16px` 内边距，悬停变深 `#005bab`，点击 `scale(0.96)`

**Secondary**：`rgba(0,0,0,0.05)` 背景，近黑文字，`4px` 圆角，`8px 16px` 内边距

**Ghost**：透明背景，近黑文字，悬停下划线

### 卡片

白色背景，`1px solid rgba(0,0,0,0.1)` 边框，`8px` 圆角，四层阴影堆叠。悬停时阴影加深。

### 输入框

白色背景，`1px solid #dddddd` 边框，`4px` 圆角，`8px` 内边距。聚焦时蓝色边框 + 微光晕。

### 标签/徽章

`#f2f9ff` 背景，`#097fe8` 文字，`9999px` 圆角，`4px 8px` 内边距，`12px` 字号 `600` 字重。

## AI Page Builder Prompt

你正在为 InkFrame 编写前端代码。该工具面向小说作者，帮助他们将小说文本自动转换为结构化剧本。

设计风格基于 Notion 设计系统：
- 主背景纯白（`#ffffff`），辅助背景暖白（`#f6f5f4`）
- 文字使用近黑色 `rgba(0,0,0,0.95)`，不用纯黑
- 唯一强调色为 `#0075de`（Notion Blue），仅用于 CTA 和链接
- 边框统一 `1px solid rgba(0,0,0,0.1)`，不用粗边框
- 按钮/输入框圆角 `4px`，卡片圆角 `8px`，大型面板/模态圆角 `12px`，徽章圆角 `9999px`
- 字体 Inter，字重 400/500/600/700 四级
- 所有 letter spacing 使用 `0`
- 阴影用四层堆叠，单层透明度不超过 0.05
- 间距基于 8px 单位
- 禁止使用 Emoji，图标用 SVG（Lucide Icons）
- 动画过渡 150-300ms，仅用于交互反馈
- 保持温暖、安静、专业的工具感，内容优先
